## МНОГОГРУНЯ — коллекция мини‑игр для 2–4 игроков
Консольная коллекция мини‑игр на Python. Поддерживает 2 или 4 игроков в зависимости от выбранной игры.

## Состав коллекции
1. «Угадай число» — игроки по очереди угадывают загаданное компьютером число. Побеждает тот, кто угадает за меньшее число попыток.
2. «Быки и коровы» — игра с угадыванием числа: игроки получают подсказки о правильности цифр и их позиций.
3. «Математический тест» — викторина с математическими примерами. Игроки зарабатывают очки за правильные ответы.
4. «Угадай слово» — игроки угадывают загаданное слово по буквам.
5. «Камень‑ножницы‑бумага» — классическая игра на выбор символа. камень > ножницы > бумага > камень.
6. «Крестики‑нолики» — классическая игра на поле 3×3 для двух игроков. Цель — выстроить 3 символа в ряд.
7. «Числовая последовательность» — игра на память: запоминайте и повторяйте последовательность чисел. С каждым уровнем длина растёт.
8. «Память на цвета» — игра на запоминание последовательности цветов.

## Используемые библиотеки
* **random** — генерация случайных чисел, слов, последовательностей.
* **time** — паузы между действиями,json**json** — сохранение и загрузка os* **os** — работа с файловой системой.
Все библиотеки входят в стандартную поставку дополнительная установка не требуется.

## Как запустить
1. Убедитесь, что у вас установлен Python 3.8+ (проверьте командой python --version или python3 --version)
2. Откройте папку с проектом и чистую командную строку
3. Напишите в терминал python main.py
4. Играйте
